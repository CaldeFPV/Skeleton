## What's in the download?
The download includes Skeleton's CSS, Normalize CSS as a reset, a sample favicon, and an index.html as a starting point.

```
Skeleton/
├── css/
│   ├── normalize.min.css
│   └── skeleton.css
├── examples/
│   └── landing/
│   		├── css
│   		├── images
│   		└── index.html
├── images/
│   └── favicon.ico
└─── index.html
```
